Data Structures and Algorithms Quiz

1. What is the time complexity of accessing an element in an array?

Correct Answer: O(n)

2. Which data structure uses Last In First Out (LIFO) order?

Correct Answer: Stack

3. What does FIFO stand for in the context of data structures?

Correct Answer: First In First Out

4. What is the time complexity of searching in a balanced binary search tree (BST)?

Correct Answer: O(log n)

5. Which search algorithm works efficiently on a sorted array?

Correct Answer: Binary Search
